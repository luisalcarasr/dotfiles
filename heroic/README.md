# Heroic Games Launcher - GitHub Dark Theme

Este paquete stow contiene un tema personalizado GitHub Dark para Heroic Games Launcher.

## Instalación

1. Instala el paquete con stow desde el directorio `.dotfiles`:
   ```bash
   cd ~/.dotfiles
   stow heroic
   ```

2. Abre Heroic Games Launcher

3. Ve a Settings > Accessibility

4. En "Custom Themes Folder", selecciona la ruta: `~/.config/heroic/themes`

5. En "Select Theme", elige "github-dark"

6. El tema se aplicará automáticamente

## Colores del Tema

Este tema usa la paleta de colores oficial de GitHub Dark:

- **Fondo principal**: #0d1117
- **Fondo secundario**: #161b22
- **Texto principal**: #e6edf3
- **Acento (azul)**: #58a6ff
- **Éxito (verde)**: #3fb950
- **Peligro (rojo)**: #da3633
- **Advertencia (amarillo)**: #d29922

## Características

- Paleta de colores consistente con GitHub Dark
- Contraste optimizado para legibilidad
- Botones y elementos de UI temáticos
- Soporte para todos los estados de AntiCheat
- Estilos personalizados para diálogos, menús contextuales y tarjetas de juegos
